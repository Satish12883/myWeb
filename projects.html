1.File Upload and User Management System

Overview:

In this project, I developed a user-friendly web application using HTML, CSS, JavaScript, and .NET that allows users to create accounts, log in, and upload files. The application focuses on providing a seamless experience for file management while ensuring user authentication and data security.

Features:
  • User Registration: Users can create new accounts by providing essential information, which is validated to ensure data integrity.
  • User Login: Implemented secure authentication, allowing users to log in with their credentials. Error handling is included to inform users of incorrect login attempts.
  • File Upload: Users can upload files easily. The application supports various file types and ensures that files are securely stored on the server.
  • Responsive Design: The interface is designed to be responsive, providing a smooth experience on both desktop and mobile devices.
Technologies Used
  • Front-End: HTML, CSS, JavaScript
  • Back-End: .NET Framework
  • Database: MySql
  • Additional Libraries/Frameworks: jQuery,Bootstrap

Conclusion:

This project showcases my ability to integrate front-end and back-end technologies to create a functional web application. It highlights my skills in user interface design, user experience, and web development best practices.

2: Automated Data Extraction from EHR Websites

Overview:

In this project, I developed an automated data extraction system using Selenium with C# to extract data from multiple Electronic Health Record (EHR) websites. The extracted data was collected, processed, and stored into MySQL tables. Furthermore, the data was transferred to a production database used by internal websites, ensuring seamless integration and access to up-to-date information.

Features:
  • Automated Data Extraction: Used Selenium WebDriver with C# to automate the process of navigating through EHR websites and extracting necessary data.
  • Data Storage: Extracted data was stored in MySQL tables, ensuring efficient data retrieval and management.
  • Data Transfer: Developed a process to transfer data from the staging MySQL tables to the production database, maintaining consistency and accuracy.
  • Error Handling: Implemented robust error handling to manage any failures during the extraction or data transfer processes.
  • Data Transformation: Ensured the data was correctly formatted before it was moved into the production environment to be used by other website databases.
Technologies Used

Automation Framework: Selenium WebDriver with C#

Database: MySQL (for both staging and production databases)

Challenges Overcome

I encountered challenges such as dealing with inconsistent website structures and dynamic content loading, which required me to implement advanced techniques for web scraping. Additionally, ensuring the integrity of the data during the transfer to the production database was critical, and I implemented strategies to mitigate potential issues.

Conclusion:

This project demonstrates my proficiency in web scraping, data processing, and database management. By automating the extraction and transfer of EHR data, I improved operational efficiency and reduced manual data entry efforts. This project highlights my skills in C#, Selenium, and MySQL, as well as my ability to integrate different systems for streamlined data management.
